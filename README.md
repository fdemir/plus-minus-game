# Artı ve Eksi Oyunu

Bilgisayar rakamları farklı n basamaklı gizli bir sayı uydurur, oyuncu + ve – bilgilerinden yola çıkarak gizli sayıyı bulmaya çalışır.

## Yükleme

```
pnpm install
```

## Projeyi Başlatmak

```
pnpm run dev
```
